# Sloshy
Sloshy is a stablecoin arbitrage product powered by Rain Protocol.

## Development setup

### Nix Shell

Install the nix shell if you haven't already.

```
curl -L https://nixos.org/nix/install | sh
```

Drop into a nix-shell.

```
cd rain-toolkit-gui
nix-shell
```

Then start the app
```
dev
```
